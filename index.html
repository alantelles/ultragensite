Sobre

UltraGen é uma linguagem simples de fácil deploy para criar sites que foca em processamento de templates sem deixar em falta recursos para obter dados de um banco de dados ou através de uma API REST.

Features

  • Fácil de aprender
  • Utilidades comuns de desenvolvimento web embutidas
  • Flexibilidade para desenvolver
  • Outros usos para processamento de texto

Sintaxe

Hello, World!

# O "#" define um comentário de uma linha
# Abaixo temos um suposto "nome do script"
# Scripts UltraGen utilizam a extensão .ultra
# hello.ultra
print: "Hello, World!"

Variáveis

# vars.ultra
a = "Eu sou uma variável"

Interpolação em texto plano

# interpolation.ultra.txt
# Quando não se utiliza a extensão .ultra o script é identificado como texto plano
# E você pode utilizar uma sintaxe especial para embutir código UltraGen no texto
Esta é uma linha de texto plano
@inter = "Este texto aparecerá por 'interpolação'"
O caractere "@" declara uma linha de código dentro do texto plano
E o resultado é: {{ inter }}
###
OBS: 3 "#" iniciam um bloco de comentário
E aqui o que será impresso: 
Esta é uma linha de texto plano
O caractere "@" declara uma linha de código UltraGen dentro do texto plano
E o resultado é: Este texto aparecerá por 'interpolação'
OBS: 3 "#" e um "!" finalizam um bloco de comentário
Você pode salvar este arquivo gerado executando "ultragen interpolation.ultra.txt --persist"
###!

Iterando por listables

list = "alguns,itens,separados,por,vírgula"
for: list, l
    # l é a variável de controle e o índice pode ser acessado por l.i
    print: "item ", l.i, ": ", l
endFor
###
item 0: alguns
item 1: itens
item 2: separados
item 3: por
item 4: vírgula
###!

unordered = "foo|bar|zika|mahailo|baholeros"
# Especificando o separador no for
# Declarando a ordenação da iteração
for: unordered, l, "|", "ASC"
    print: "item ", l.i, ": ", upper(l)
endFor
###
item 0: BAHOLEROS
item 1: BAR
item 2: FOO
item 3: MAHAILO
item 4: ZIKA
###!
